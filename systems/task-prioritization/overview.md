# Task Prioritization System

## Problem

In my daily work, I often struggled to decide what to work on first.

Everything felt important and urgent.  
I would end up working on tasks that were easy or visible instead of tasks that actually mattered.

This led to:

- Important work getting delayed  
- Low-impact tasks taking more time  
- Feeling busy but not productive  

---

## Why This Happens

- No clear decision-making system  
- Lack of visibility on task importance  
- Reacting to tasks instead of planning  

---

## Outcome

This system helps me:

- Identify what truly matters  
- Focus on high-impact tasks  
- Make better decisions on what to do next
